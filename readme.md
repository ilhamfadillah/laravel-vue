## How To Install In Ubuntu

- rename .env.example to .env
- settting how to laravel can access your database in DB section
- open your terminal
- composer install
- npm install
- npm run dev 
- php artisan serve
- npm run watch (in new tab terminal) 
- now you can run in localhost:8000
